# Menberemariyam - Personal Portfolio

A modern, responsive portfolio website for a Front-End and Android Developer.

## Features

- **Responsive Design**: Works on all devices from mobile to desktop
- **Interactive Elements**: Animated components, sliders, and modals
- **Form Validation**: Contact form with real-time validation
- **Smooth Animations**: CSS transitions and JavaScript animations
- **Modern UI**: Clean design with vibrant colors

## Sections Included

1. **Hero Section** - Introduction with code snippet
2. **About Section** - Skills and expertise
3. **Projects Section** - Featured work with details
4. **Testimonials Section** - Client reviews with slider
5. **Gallery Section** - Development showcase
6. **Videos Section** - Tutorials and learning content
7. **Contact Section** - Contact form and information
8. **Footer** - Links and newsletter subscription

## Setup Instructions

1. Download all files into a folder
2. Open `index.html` in any modern web browser
3. No server or build process required - works directly in browser
