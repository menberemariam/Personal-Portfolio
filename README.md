# 👋 Menberemariam Teshome Asrat - Portfolio

A modern, advanced portfolio website showcasing my work as a Full-Stack Software Engineer. Built with a modular architecture for better maintainability and scalability.

## ✨ New Modular Architecture

This portfolio has been refactored into a clean, modular structure:

```
📁 Portfolio
├── 📁 css/              # Modular CSS files
│   ├── main.css         # Main CSS (imports all modules)
│   ├── variables.css    # Theme & variables
│   ├── base.css         # Reset & base styles
│   ├── animations.css   # Animations
│   ├── components.css   # Reusable components
│   ├── navigation.css   # Navigation styles
│   ├── hero.css         # Hero section
│   ├── sections.css     # Section styles
│   ├── contact.css      # Contact form
│   ├── footer.css       # Footer styles
│   └── responsive.css   # Media queries
│
├── 📁 js/               # Modular JavaScript files
│   ├── main.js          # Main entry point
│   ├── navigation.js    # Navigation logic
│   ├── testimonials.js  # Testimonials slider
│   ├── modals.js        # Modal management
│   ├── contact-form.js  # Form validation
│   └── utils.js         # Utility functions
│
├── 📁 images/           # Project images
├── index.html           # Main HTML file
└── README.md            # This file
```

## 🎨 Modular Architecture Benefits

### CSS Modules
- **variables.css**: Centralized theme configuration
- **base.css**: Reset and base styles
- **animations.css**: All keyframe animations
- **components.css**: Reusable UI components
- **navigation.css**: Navigation bar styles
- **hero.css**: Hero section styles
- **sections.css**: Section-specific styles
- **contact.css**: Contact form styles
- **footer.css**: Footer styles
- **responsive.css**: All media queries

### JavaScript Modules
- **main.js**: Application entry point
- **navigation.js**: Navigation & scroll effects
- **testimonials.js**: Testimonials slider logic
- **modals.js**: Video & project modals
- **contact-form.js**: Form validation & submission
- **utils.js**: Utility functions & helpers

## 🔧 Customization

### Adding New Styles
1. Create a new CSS file in `css/` folder
2. Import it in `css/main.css`

### Adding New JavaScript Functionality
1. Create a new JS module in `js/` folder
2. Export a class with your functionality
3. Import and initialize in `js/main.js`
