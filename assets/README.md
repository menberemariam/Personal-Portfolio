# Assets Folder

This folder contains downloadable assets for the portfolio.

## Files

- **Menberemariyam-Teshome-CV.pdf** - Your professional CV/Resume

## Instructions

To update your CV:
1. Replace the `Menberemariyam-Teshome-CV.pdf` file with your latest CV
2. Keep the same filename or update the filename in `js/utils.js`
3. The CV will be automatically downloadable from the portfolio

## Supported Formats

- PDF (recommended)
- DOCX
- DOC

## File Size

Keep the CV file size under 5MB for optimal download performance.
