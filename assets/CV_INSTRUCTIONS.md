## File Naming Convention

The download button is configured to look for:
- **File location:** `assets/Menberemariyam-CV.pdf`
- **Downloaded as:** `Menberemariyam_Teshome_CV.pdf`

## Alternative: Use a Different Filename

If you want to use a different filename, update line 27 in `js/utils.js`:

```javascript
link.href = 'assets/YOUR-FILENAME.pdf';
link.download = 'YOUR-DOWNLOAD-NAME.pdf';
```

## Troubleshooting

**CV not downloading?**
- Check that the file is named exactly: `Menberemariyam-CV.pdf`
- Check that it's in the `assets/` folder
- Check browser console for errors
- Try refreshing the page (Ctrl + F5)

**Wrong file downloading?**
- Clear browser cache
- Check the filename in `js/utils.js`
- Verify the file path is correct

## Creating Your CV PDF

If you need to create a PDF from your CV:

### Option 1: Microsoft Word
1. Open your CV in Word
2. File → Save As
3. Choose "PDF" as file type
4. Save as `Menberemariyam-CV.pdf`

### Option 2: Google Docs
1. Open your CV in Google Docs
2. File → Download → PDF Document
3. Rename to `Menberemariyam-CV.pdf`

### Option 3: Online CV Builders
- [Canva](https://www.canva.com/create/resumes/)
- [Resume.io](https://resume.io/)
- [Novoresume](https://novoresume.com/)
