# 📄 Convert Your CV HTML to PDF

## Quick Method (Recommended)

### Option 1: Using Your Browser (Easiest!)

1. **Open the CV HTML file:**
   - Navigate to: `assets/Menberemariyam-CV.html`
   - Right-click → Open with → Your browser (Chrome, Edge, Firefox)

2. **Print to PDF:**
   - Press `Ctrl + P` (Windows) or `Cmd + P` (Mac)
   - In the print dialog:
     - **Destination:** Select "Save as PDF" or "Microsoft Print to PDF"
     - **Layout:** Portrait
     - **Margins:** None or Minimal
     - **Background graphics:** Check this box (important!)
   - Click "Save" or "Print"
   - Save as: `Menberemariyam-CV.pdf` in the `assets/` folder

3. **Done!** Your CV is now ready for download

### Option 2: Using Microsoft Edge (Best Quality)

1. Open `assets/Menberemariyam-CV.html` in Microsoft Edge
2. Click the three dots (⋯) → Print
3. Select "Save as PDF"
4. Click "Save"
5. Name it: `Menberemariyam-CV.pdf`
6. Save in the `assets/` folder

### Option 3: Using Google Chrome

1. Open `assets/Menberemariyam-CV.html` in Chrome
2. Press `Ctrl + P`
3. Destination: "Save as PDF"
4. More settings:
   - Paper size: A4
   - Margins: None
   - Scale: 100%
   - Background graphics: ✓ Checked
5. Click "Save"
6. Save as `Menberemariyam-CV.pdf` in `assets/` folder

## Alternative Methods

### Option 4: Online PDF Converter

1. Go to: https://www.html2pdf.com/ or https://pdfcrowd.com/
2. Upload `Menberemariyam-CV.html`
3. Click "Convert"
4. Download the PDF
5. Rename to `Menberemariyam-CV.pdf`
6. Move to `assets/` folder

### Option 5: Using Microsoft Word

1. Open Microsoft Word
2. File → Open → Select `Menberemariyam-CV.html`
3. File → Save As
4. Choose "PDF" as file type
5. Save as `Menberemariyam-CV.pdf` in `assets/` folder

## Troubleshooting

### PDF looks wrong?
- Make sure "Background graphics" is checked when printing
- Try a different browser
- Adjust margins to "None"

### File too large?
- Use online PDF compressor: https://www.ilovepdf.com/compress_pdf
- Reduce to under 2MB

### Download still not working?
- Clear browser cache (Ctrl + Shift + R)
- Check browser console (F12) for errors
- Verify filename is exactly: `Menberemariyam-CV.pdf`
